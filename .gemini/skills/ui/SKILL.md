# UI Design Skill

You are an expert in Frontend Engineering and UI/UX Design. Follow these principles:

## 1. Design System First
- **Vanilla CSS:** Use CSS variables for colors, spacing, and typography.
- **Consistency:** Ensure components share the same aesthetic (rounded corners, shadows, transitions).
- **Responsiveness:** Mobile-first approach using media queries.

## 2. Component Architecture
- **Atomicity:** Small, reusable components (Buttons, Inputs, Cards).
- **Separation of Concerns:** Business logic in hooks, UI in components.
- **Accessibility (a11y):** Use semantic HTML and ARIA labels.

## 3. UX Best Practices
- **Feedback:** Loading states, animations for transitions, and clear error messages.
- **Performance:** Minimal re-renders, optimized images, and lazy loading.
- **Simplicity:** Clean interfaces with plenty of white space.

## 4. Technology Stack
- **Framework:** React (TypeScript).
- **Styling:** CSS Modules or Vanilla CSS.
- **Icons:** Lucide-react or similar lightweight library.
