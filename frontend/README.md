# Preacher Studio Frontend

The web interface for Preacher Studio, a homiletic mentoring platform.

## Features
- **AI Mentorship:** Real-time suggestions for sermon improvement.
- **Sermon Management:** Create, list, and edit sermons.
- **Export Dashboard:** Download sermons in PDF and Word formats.
- **Auth System:** Secure login and registration.
- **PWA Support:** Installable web app.
- **Responsive Design:** Optimized for mobile and desktop.

## Tech Stack
- **Framework:** React + TypeScript + Vite
- **State:** Zustand
- **API:** Axios
- **Styling:** Vanilla CSS (CSS Variables)

## Setup
1. `cd frontend`
2. `npm install`
3. `npm run dev`

## Build
`npm run build`
