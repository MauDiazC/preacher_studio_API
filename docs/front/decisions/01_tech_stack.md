# Tech Stack Selection

## Decision: Vite + React + TypeScript
- **Rationale:** Standard choice for modern, fast frontend development. TypeScript provides type safety which is crucial for interacting with the backend API.

## Decision: Vanilla CSS Variables
- **Rationale:** Avoids the overhead of external frameworks (like Tailwind or Bootstrap) and allows for a truly custom, lightweight design system.

## Decision: Zustand
- **Rationale:** Simpler and more lightweight than Redux, with a great hook-based API that fits perfectly with React.

## Decision: Axios
- **Rationale:** Provides built-in support for interceptors, which are essential for handling JWT tokens and refreshing sessions.
